# Your interactive and bindable Audio Mixer

## Using: 

- [Pycaw](https://github.com/AndreMiras/pycaw)
- [Tkinter Slider](https://www.pythontutorial.net/tkinter/tkinter-slider/)
- [Possible Audio Library: ossaudiodev](https://docs.python.org/3/library/ossaudiodev.html#module-ossaudiodev)

- [Possible Audio Library: sounddevice](https://python-sounddevice.readthedocs.io/en/0.4.5/installation.html)
